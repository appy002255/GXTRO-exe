# GXTRO 下載工具

## 專案簡介

GXTRO 媒體下載工具是一個基於 `yt-dlp`、`PyQt5` 和 `VLC` 的圖形化介面應用程式，旨在提供一個方便易用的影片和直播下載及簡易剪輯工具。本工具支援多種影音平台，並內建了影片預覽、時間/空間裁剪、浮水印、背景音樂等功能。

**如果你是從 GitHub Releases 或其他來源下載的 exe 免安裝版，無需安裝 Python 或任何依賴，直接執行 `GXTRO.PIYEN.DOWNLOADER.exe` 即可使用！**

---

## 功能特色

*   支援多平台影音下載 (YouTube, Bilibili, Twitter, Facebook, Instagram, TikTok, Twitch, Vimeo 等)
*   支援部分平台直播下載
*   圖形化使用者介面 (PyQt5)，支援暗色主題
*   內建畫質/格式選擇、影片封面預覽、簡潔/詳細日誌切換
*   LOG 簡潔模式下 GUI 也完整顯示所有訊息
*   影片剪輯功能：時間裁剪、空間裁剪、解析度調整、播放速度調整
*   整合 VLC 播放器，支援影片預覽、截圖、音量控制、播放速度調整
*   支援浮水印添加 (自訂位置、大小、透明度)
*   支援背景音樂添加 (混音、音量平衡)
*   音訊提取功能
*   螢幕截圖功能
*   自動清理與擷取網址，支援複製雜訊內容
*   支援 AI 字幕自動生成（Whisper 模型）
*   支援多檔案合併、批次處理

# GitHub 軟體安裝器

這是一個簡單的命令列工具，用於從 GitHub 下載並安裝發布的軟體。

## 功能特點

- 自動獲取最新的發布版本
- 支援下載和解壓縮 zip 檔案
- 顯示下載進度條
- 支援自定義安裝目錄

## 安裝需求

1. Python 3.6 或更高版本
2. 安裝必要的依賴包：

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python installer.py <GitHub用戶名> <倉庫名稱> <安裝目錄>
```

例如：

```bash
python installer.py username my-software C:\Program Files\MySoftware
```

## 注意事項

1. 確保您有足夠的權限訪問目標安裝目錄
2. 如果下載的是 zip 檔案，安裝器會自動解壓縮
3. 如果遇到權限問題，請以管理員身份運行命令提示字元

## 錯誤處理

如果遇到問題，請檢查：

1. GitHub 倉庫是否存在
2. 是否有發布版本
3. 網路連接是否正常
4. 目標目錄是否有寫入權限
